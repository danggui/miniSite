代码：

<h1>横向排版</h1>
<ul class="form-item-list">
<li class="form-item-li form-item-li-noborder">
<span class="form-label split-h-label-w pull-left">公司</span>
<div class="form-control split-h-control-w">
<input class="" type="text" name="company" id="company">
</div>
</li>
<li class="form-item-li form-item-li-noborder">
<span class="form-label split-h-label-w pull-left">用户名</span>
<div class="form-control split-h-control-w">
<input class="" type="text" name="username" id="username">
</div>
</li>
<li class="form-item-li form-item-li-noborder">
<span class="form-label split-h-label-w pull-left">密码</span>
<div class="form-control split-h-control-w">
<input  class="" id="pwd" type="password" value="" name="pwd">
</div>
</li>
<li class="form-item-li form-item-li-noborder gap-v-sm">
<div class="form-control">
<span class="pull-left">
<label class="check"><input id="remerber" type="checkbox" checked=""><i></i><span>保存密码</span></label>
</span>
<a class="pull-right">忘记密码？</a>
</div>
</li>
<li class="form-item-li btn-v-group">
<button id="btn_enter">登录</button>
</li>
</ul>

说明：

  • 用ul,li做包裹,ul加入form-item-list样式类,li加入form-item-li（默认有下边框,如需去除需加入form-item-li-noborder样式类）
  • 横向分布用span做标题需加入form-label,split-h-label-w,pull-left样式类，用div包含input，button需加入form-control,split-h-control-w样式类
  • 如有必填需要，需在包含input，button的div中加form-required样式类
  • li之前的间距可以使用gap-v-sm,gap-v样式类做调控
  • 按钮组自适应需在Li标签加上btn-v-group或则btn-h-group样式类

  • 如有必填需要，需在包含input，button的div中加form-required样式类

  • 表单组合需要嵌套ul,li ul的加入form-group-nospacing样式类
