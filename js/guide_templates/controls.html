按钮

未分组

行分组

列分组

代码：

<h3>未分组</h3>
<div class="btn-bar">
<button type="button">普通的按钮</button>
</div>
<h3>行分组</h3>
<ul class="btn-h-group">
<li class="flex-btn"><button type="button" class="btn-gray">低调的按钮</button></li>
<li class="flex-btn"><button type="button" class="btn-light-gray">忧郁的按钮</button></li>
</ul>
<h3>列分组</h3>
<div class="btn-v-group gap-v-sm">
<button type="button">普通的按钮</button>
<button type="button" class="btn-gray">低调的按钮</button>
<button type="button" class="btn-link">白白的按钮</button>
<button type="button" class="btn-white">宛若新生的按钮</button>
<button type="button" class="btn-lg">肥胖的按钮</button>
<button type="button" class="btn-sm">精妙的按钮</button>
<button type="button" class="btn-cute">可爱的按钮</button>
</div>

说明：

  • 默认按钮根据内容定义宽度
  • 行分组，按钮组自动分配宽度,需要用ul,li做包裹，li的class需要flex-btn样式
  • 列分组，按钮组自适应组宽度

文本框

行分组

列分组

代码：

<h3>行分组</h3>
<p class="btn-h-group">
<input type="text" value="文字示例">
<input type="text" disabled value="被禁用的文字示例">
</p>
<h3>列分组</h3>
<p class="btn-v-group">
<input type="text" value="写错了的文字示例" class="has-error">
<input type="text" disabled value="写错了还不能改的文字示例" class="has-error">
</p>

说明：

  • 错误的样式需在文本框下加.has-error

多行文本框

代码：

<textarea></textarea>
<textarea class="has-error"></textarea>

说明：

  • 错误的样式需在多行文本框下加.has-error

checkbox

代码：

<label class="check"><input type="checkbox" checked><i></i><span>选择项 1</span></label>
<label class="check"><input type="checkbox" disabled><i></i><span>禁用</span></label>
<label class="check"><input type="radio" checked><i></i><span>选择项 1</span></label>
<label class="check"><input type="radio" disabled><i></i><span>禁用</span>

说明：

  • 选择项每条都用 label.check 打包，文字部分用 span 打包，文字前需加入否则无法显示出样式。
  • checked 属性不要写成 checked=true 或 checked=checked。
